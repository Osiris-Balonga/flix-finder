## Features

Personalized Movie Suggestions: Discover movies based on your favorite genres and preferences.
Search Functionality: Easily find movies by title or keywords.
Favorites List: Save movies to your favorites and access them anytime.
Movie & Cast Info: Get detailed information about movies, actors, and their filmography.
Watch Trailers: View trailers to get a sneak peek before you dive in.
Development
To start developing locally:

bash
Copier le code
pnpm install
pnpm dev
Building
To create a production-ready build:

bash
Copier le code
pnpm build
The built files will be output to the `/dist`
folder.

## VERSION FRANÇAISE

FlixFinder est votre application incontournable pour découvrir des films adaptés à vos préférences. Alimentée par l'API TMDB, elle vous propose des recommandations personnalisées, vous permet de rechercher des films, de les ajouter à vos favoris et offre des informations détaillées sur les films et leurs acteurs. Que vous cherchiez votre prochain film préféré ou que vous souhaitiez explorer de nouveaux genres, FlixFinder est là pour vous !

## Fonctionnalités

Suggestions de films personnalisées : Découvrez des films basés sur vos genres et préférences favoris.
Fonctionnalité de recherche : Trouvez facilement des films par titre ou mots-clés.
Liste de favoris : Enregistrez vos films préférés et accédez-y à tout moment.
Infos sur les films et acteurs : Obtenez des informations détaillées sur les films, les acteurs et leur filmographie.
Regarder des bandes-annonces : Visionnez les bandes-annonces pour un aperçu avant de commencer.
Développement
Pour démarrer le développement localement :
